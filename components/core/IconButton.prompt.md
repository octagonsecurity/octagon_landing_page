Square, icon-only button for toolbars, nav, and compact actions. Pass an SVG icon as children; always set `label`.

```jsx
<IconButton label="Call" variant="primary"><LucidePhone/></IconButton>
<IconButton label="Menu"><LucideMenu/></IconButton>
```

Variants: `secondary` (outlined, default), `primary` (red), `dark`. Sizes `sm | md | lg`.
